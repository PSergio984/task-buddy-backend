# Integrations

**Analysis Date:** 2026-05-08

## Databases

**PostgreSQL:**
- **Purpose:** Primary production database.
- **Library:** `psycopg2-binary`, `sqlalchemy`.
- **Config:** `DATABASE_URL` in `.env`.
- **Migrations:** Managed by Alembic.

**SQLite:**
- **Purpose:** Local development and automated testing.
- **Library:** `aiosqlite`, `databases[sqlite]`.
- **Config:** `sqlite:///./test.db`.

**Redis:**
- **Purpose:** Message broker for Celery and caching.
- **Config:** `REDIS_URL` in `.env`.
- **Library:** `redis`.

## Background Processing

**Celery:**
- **Purpose:** Handling asynchronous tasks like email dispatch and periodic cleanups.
- **Broker:** Redis.
- **Config:** Configured in `app/celery_app.py`.

## External Services

**Brevo (formerly Sendinblue):**
- **Purpose:** Transactional email delivery (OTP, password reset, notifications).
- **Type:** SMTP / REST API.
- **Config:** `DEV_MAIL_API_KEY`, `DEV_MAIL_URL`, `DEV_MAIL_SMTP_*`.
- **Location:** `app/tasks.py` (email dispatch logic).

**Sentry:**
- **Purpose:** Error tracking and performance monitoring.
- **Library:** `sentry-sdk`.
- **Config:** `SENTRY_DSN` in `.env`.
- **Initialization:** Typically in `app/main.py`.

## Security & Authentication

**JWT (JSON Web Tokens):**
- **Purpose:** Stateless user authentication.
- **Algorithm:** HS256 (configured via `ALGORITHM` in `.env`).
- **Library:** `python-jose`.
- **Flow:** Issued on login, verified via `app/api/dependencies.py` or similar.

**Argon2:**
- **Purpose:** Secure password hashing.
- **Library:** `passlib[argon2]`, `argon2-cffi`.

## APIs & Protocols

**REST API:**
- **Type:** OpenAPI / Swagger.
- **Documentation:** Automatically generated by FastAPI at `/docs` (Swagger UI) and `/redoc`.
- **CORS:** Configured in `app/main.py` via `CORSMiddleware`.
