# Plan: 01-01 ORM Migration & AsyncSession Setup

**Goal**: Transition from SQLAlchemy Core/databases to SQLAlchemy 2.0 ORM with AsyncSession.

## Proposed Changes

### 1. Relocate Pydantic Models
- **[MOVE]** `app/models/*.py` → `app/schemas/*.py`
- Update `__init__.py` in both directories.
- Update imports in:
  - `app/api/routers/*.py`
  - `app/security.py`
  - `app/internal/audit.py`
  - `app/tasks.py`

### 2. Configure Async Database
- **[MODIFY]** `app/database.py`
  - Remove `databases` library.
  - Add `create_async_engine`, `async_sessionmaker`.
  - Export `AsyncSession` type hint.
- **[MODIFY]** `app/dependencies.py`
  - Add `get_db` dependency.

### 3. Implement ORM Models
- **[NEW]** `app/models/base.py` (DeclarativeBase)
- **[NEW]** `app/models/user.py` (User)
- **[NEW]** `app/models/task.py` (Task, SubTask)
- **[NEW]** `app/models/tag.py` (Tag, association table)
- **[NEW]** `app/models/audit.py` (AuditLog)
